Welcome to Nest Up

Nest Up is a robust backend application built with NestJS, designed to handle modern authentication, user management, and modular development.

This project includes the following core modules:

  • AuthModule: Secure login and JWT authentication
  • UserModule: Manage user data and profiles
  • EmailModule: Email verification and notifications
  • RoleModule: Role-based access control
  • CommonModule: Reusable utilities and decorators
  • SeedModule: Database seeding and initialization for development and testing
View NestJS Docs

@Auth Decorator

Protect routes easily using @Auth() in your controllers.

Make sure to import AuthModule into the module where you use it.

HttpResponse Helper

Standardize your responses using the HttpResponse utility class.

Useful when not relying on interceptors or for customized responses.

Global Exception Handler

Errors are gracefully managed by a centralized GlobalExceptionHandler.

No need to wrap try-catch in most services or controllers.
